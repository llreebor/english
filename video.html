О Курсах
  • Работаю с группами в онлайн в программе Zoom
  • В данный момент работаю с подростковыми группами Beginner (9 лет), Elementary (10-11 лет), PreIntermediate, Intermediate, Upper-Intermediate. При желании попасть в одну из продолжающих групп, пишите на почту jennyakimenko@yandex.ru, указав результаты теста и возраст ребенка. Ребенок может присоединиться к группе при наличии места в группе, согласно результатов теста
  • Набираю одну группу Elementary 12+ c полного нуля. Курс Beginner будет дан в рамках Elementary. Для записи в группу, отправьте заявку
  • Набираю две группы Elementary для взрослых с полного нуля. Выполнение домашних заданий и вовлеченность обязательны. Для записи в группу, отправьте заявку
  • Если хотите записать ребенка и начать занятия через несколько лет, запишитесь в лист ожидания

Упорным
